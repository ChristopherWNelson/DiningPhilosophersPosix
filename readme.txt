Description:
Simulates the 5 dining philosophers problem, using POSIX threads, where each 
philosopher repeats a cycle of eating and thinking untill all philosophers 
have eaten for a least 100 seconds.

How to run:
Type 'make' to compile the source code into an exe.
Type './philo' to run the exe.
